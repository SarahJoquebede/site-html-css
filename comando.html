Comandos SQL

DDL- Data Definition Language

Define e altera a estrutura do banco de dados.

CREATE TABLE alunos(...);
 ALTER TABLE alunos ADD;
DROP TABLE alunos;

DML - Data Manipulation Language

Manipula os dados dentro das tabelas.

INSERT INTO alunos VALUES (...);
UPDATE alunos SET ... WHERE ...;
DELETE FROM alunos WHERE ...;
SELECT * FROM alunos;
